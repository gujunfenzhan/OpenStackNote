bridge网桥

bridge

  • 查看所有的命名空间
ip netns list
  • 创建命名空间
ip netns add <命名空间>
  • 查看指定命名空间的信息
ip netns exec <命名空间的ID> ip a
  • 查看所有网桥
ovs-vsctl list-br

*输出ovs

ovs-vsctl show
  • 添加网桥
ovs-vsctl add-br <网桥名>
  • 删除网桥
ovs-vsctl del-br <网桥名>
  • 创建内部通信端口
ovs-vsctl add-port <网桥名> <端口名> -- set interface <端口名> type=internal
  • 将端口加入命名空间
ip link set <端口名> netns <命名空间>

ip netns exec namespace1 ip addr add 172.168.0.10/24 dev tap1
ip netns exec namespace1 ip link set tap1 up
ip netns exec namespace1 ip link set lo up

ip netns exec namespace2 ip addr add 172.168.0.20/24 dev tap2
ip netns exec namespace2 ip link set tap2 up
ip netns exec namespace2 ip link set lo up

ip netns exec namespace1 ping 172.168.0.20 -c 4
ip netns exec namespace2 ping 172.168.0.10 -c 4
