安全组

安全组

  • 列出所有安全组
nova secgroup-list

openstack security group list
  • 查看安全组规则

openstack security group show <安全组名字>

  • 创建安全组
nova secgroup-create <安全组名字> <安全组描述>

openstack security group create <安全组名字>
[--description <描述>]
  • 添加安全组规则
nova secgroup-add-rule <安全组名> <数据包协议/icmp/tcp> <源端口> <目标端口> <网络号 x.x.x.x/x>
openstack security group rule create <安全组名> 
[--src-ip <网络号 x.x.x.x/x>] 
[--dst-port <ip范围,-1指所有 x:x 1:65535>]
[--proto <数据表类型>]
  • 删除安全组规则
nova secgroup-delete-rule <安全组名字> <数据包协议/icmp/tcp> <源端口> <目标端口> <网络号 x.x.x.x/x>
  • 删除安全组
nova secgroup-delete <安全组名>

openstack security group delete <安全组名字>
  • 查看安全组详细信息
openstack security group show <安全组名>
