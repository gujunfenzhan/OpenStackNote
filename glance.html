镜像

镜像

  • 查看镜像列表

openstack image list

  • 查看镜像详细信息

openstack image show <镜像名/镜像ID>

  • 删除镜像文件

openstack image delete <镜像名/镜像ID>

  • 修改镜像属性
openstack image set <镜像名/镜像ID> 
[--name <镜像名>] #修改镜像名
[--disk-format <qcow2/...>] #磁盘格式，一般qcow2即可
[--container-format <bare/...>] #写就完事了
[--protected/--unprotected] #是否受保护的
[--public/--private] #是公共的还是私有的

glance image-update --name <新的镜像名>
[--min-ram <最小内存>]
[被修改的镜像ID]
  • 创建镜像
openstack image create <镜像名> 
[--disk-format <qcow2/...>] #磁盘格式，一般qcow2即可
[--container-format <bare/...>] #写就完事了
[--protected/--unprotected] #是否受保护的
[--public/--private] #是公共的还是私有的
[--file <镜像路径>] #镜像路径

glance image-create --name <镜像名>
[--min-disk <最小大小>]
[--min-ram <最小内存>]
[--disk-format <镜像格式>]
[--container-format <容器格式/bare>]
[--progress< <镜像路径>]
