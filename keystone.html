认证服务

认证服务

  • 创建用户
    openstack user create <用户名>
[--password <密码>] #用户登录密码
[--domain <域>] #需要看有什么域
[--enable/--disable] #用户是否激活
[--email <邮箱>] #无所谓，可写可不写
  • 删除用户

openstack user delete <用户名/用户ID>

  • 列出所有用户

openstack user list

  • 查看用户信息

openstack user show <用户名>

  • 创建项目
openstack project create <项目名>
[--domain <域>]
[--enable/--disable]
[--description <描述>]
  • 创建角色

openstack role create <角色名>

*把某用户关联绑定到对应的项目和角色

openstack role add <要把用户加入的那个角色名>
[--project <用户要参与的项目>]
[--user <要关联的用户>]
