glance和nova复习

glance&nova

  • 查看ciroos镜像详细信息

qemu-img info /opt/cirros-0......img

  • 使用glance命令上传cirros镜像包到云平台，命名为cirros，最小磁盘为0，最小内存为512M，格式为qcow2，显示上传进度。
glance image-create --name cirros --min-disk 0 --min-ram 512 --disk-format qcow2 --container-format bare --progress< /pot .....img
  • 修改镜像
glance image-update --name cirros23 --min-ram 1024 <镜像名>
  • 创建镜像类型
openstack flavor create fla23 --ram 512 --disk 2 --vcpus 1
  • 创建安全组
nova secgroup-create sec23 "sec"
  • 给安全组加规则
openstack security group rule create sec23 --proto icmp
openstack security group rule create sec23 --proto tcp --dst-port 1:65535
